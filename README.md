Aperçu
Confiance est un système de gestion bancaire complet qui permet de gérer les clients, comptes et transactions. Le projet inclut un tableau de bord moderne avec des statistiques en temps réel, une interface utilisateur responsive et des fonctionnalités complètes de CRUD (Create, Read, Update, Delete).

# Fonctionnalités

Gestion des Clients : Ajouter, visualiser, modifier et supprimer des clients
Gestion des Comptes : Créer et gérer des comptes bancaires
Transactions : Suivre et gérer les transactions financières
Tableau de Bord : Statistiques en temps réel et activités récentes
Authentification : Système d'authentification sécurisé
Interface Responsive : Design moderne adapté à tous les écrans
Recherche : Fonctionnalité de recherche avancée
Pagination : Navigation facile entre les données

# Prérequis

Serveur Web : Apache ou Nginx
PHP : Version 7.4 ou supérieure
Base de données : MySQL ou MariaDB
Extensions PHP : PDO, MySQLi
Outils : Git, Composer (optionnel)
